# 노동요 커버 검정 마스크 방향 반전 작업 보고서

## 원래 요청

이전·다음 커버의 검정 그라데이션이 화면 바깥에서 안쪽으로 들어오는 방향이 아니라, 현재곡이 있는 중심에서 화면 바깥으로 퍼지는 방향이 되게 변경한다.

## 완료한 작업

- 왼쪽의 이전곡과 전전곡 마스크 방향을 `90deg`에서 `270deg`로 변경했다.
- 왼쪽 커버는 현재곡과 맞닿는 오른쪽 가장자리에서 검정 효과가 시작해 왼쪽 바깥 방향으로 사라진다.
- 오른쪽의 다음곡과 다다음곡 마스크 방향을 `270deg`에서 `90deg`로 변경했다.
- 오른쪽 커버는 현재곡과 맞닿는 왼쪽 가장자리에서 검정 효과가 시작해 오른쪽 바깥 방향으로 사라진다.
- 다음 곡 전환 중 현재 커버가 왼쪽 슬롯으로 이동할 때도 `270deg` 방향을 적용했다.
- 이전 곡 전환 중 현재 커버가 오른쪽 슬롯으로 이동할 때도 `90deg` 방향을 적용했다.
- 마스크 강도, 커버 위치, 겹침, 크기, 평면 이동 애니메이션은 변경하지 않았다.

## 기존 기능에 미치는 영향

- CSS 마스크 방향 변수 여섯 곳만 변경했다.
- HTML과 JavaScript는 변경하지 않았다.
- YouTube 재생, 이어듣기, 이전·다음 곡 선택, 재생바 조절 로직에는 영향이 없다.

## 변경 파일

- `styles.css`: 좌우 일반 상태와 전환 상태의 검정 마스크 방향 반전
- `current_task.md`: 이번 작업 보고서로 갱신
