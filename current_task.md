# 노동요 재생바 호버 시간 표시 작업 보고서

## 원래 요청

- 노동요 재생바에 마우스를 올리면 해당 위치와 시간 숫자를 표시한다.
- 재생바를 클릭하면 표시된 위치로 곡을 이동한다.
- 재생바 양쪽 시간 숫자를 재생바 가까이 붙인다.
- 전체 진행 영역은 좌우 테두리에서 10px 떨어지게 배치한다.

## 완료한 작업

### 호버 위치와 시간 표시

- 재생바를 감싸는 전용 요소를 추가했다.
- 포인터 위치를 재생바의 백분율로 계산한다.
- 해당 위치에 흰색 점과 `분:초` 형식의 시간 툴팁을 표시한다.
- 재생 시간이 없거나 플레이어가 준비되지 않은 경우 툴팁을 표시하지 않는다.
- 포인터가 재생바를 벗어나면 툴팁을 숨긴다.

### 클릭 및 드래그 이동

- 기존 range 입력의 `input` 이벤트와 YouTube 플레이어 `seekTo` 경로를 유지했다.
- 따라서 재생바 클릭과 드래그 모두 선택한 위치의 시간으로 이동한다.
- 호버 시간은 실제 플레이어 재생 시간을 우선 사용하고, 저장된 곡 길이를 보조 값으로 사용한다.

### 간격 조정

- 진행 영역의 최대 700px 너비 제한을 제거하고 플레이어 영역 전체 폭을 사용하도록 변경했다.
- 플레이어 영역 자체가 좌우 10px 여백을 가지고 있어 시간 숫자도 양쪽 테두리에서 10px 떨어진다.
- 시간 숫자 열을 내용 너비에 맞추고 재생바와의 간격을 데스크톱 6px, 모바일 5px로 줄였다.
