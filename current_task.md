# 노동요 Cover Flow 간격 및 디자인 통일 작업 보고서

## 원래 요청

화면 폭이 좁아져도 이전·현재·다음 곡 커버의 간격이 흐트러지지 않고 항상 같은 간격으로 보이게 한다. 노동요 플레이어의 디자인을 달력 디자인과 통일하고, 이전·다음 곡 커버는 투명도를 낮추는 대신 검정 그라데이션으로 뒤에 있는 느낌을 표현한다.

## 완료한 작업

### Cover Flow 간격 고정

- 카드 너비의 퍼센트로 위치를 이동하던 방식을 제거했다.
- 중앙, 좌우 한 칸, 좌우 두 칸 커버의 중심 좌표를 공통 좌표계로 정의했다.
- 중앙에서 한 칸까지와 한 칸에서 두 칸까지의 중심 간격을 같은 값으로 맞췄다.
- 컨테이너 너비를 기준으로 커버 크기, 간격, 높이가 하나의 동일한 비율로 함께 축소되게 했다.
- 모바일 전용으로 간격을 다시 덮어쓰던 CSS를 제거해 화면 크기에 따라 배치 비율이 달라지지 않게 했다.
- 다음·이전 곡 전환 애니메이션도 같은 고정 좌표를 사용하도록 변경했다.

### 뒤쪽 커버 깊이 표현

- 이전·다음 커버에 적용하던 `opacity`와 밝기·채도 `filter`를 제거했다.
- 커버 이미지 자체는 불투명하게 유지한다.
- 좌우 한 칸 커버에는 바깥쪽이 더 어두운 검정 그라데이션을 적용했다.
- 좌우 두 칸 커버에는 더 진한 검정 그라데이션을 적용해 뒤쪽 깊이를 표현했다.
- 곡 전환 중에는 커버가 이동하는 슬롯에 맞춰 검정 그라데이션도 함께 전환된다.

### 달력 디자인과 통일

- 노동요 플레이어 배경을 달력 주간 카드와 같은 `#242424`로 변경했다.
- 라운드를 달력 카드와 같은 `14px`로 맞췄다.
- 구분선은 달력 날짜 셀 계열의 `#3b3b3b`을 사용했다.
- 달력 주간 카드와 같은 아래쪽 적층 그림자 값을 적용했다.

## 기존 기능에 미치는 영향

- HTML 구조와 JavaScript는 변경하지 않았다.
- YouTube 플레이어 생성, 재생, 이전·다음 곡 선택, 이어듣기, 재생바 조절 로직은 변경하지 않았다.
- 기존 애니메이션 클래스 이름과 420ms 전환 흐름은 그대로 유지하고 CSS의 위치 계산과 시각 표현만 변경했다.

## 변경 파일

- `styles.css`: Cover Flow 공통 축척 좌표, 균일 간격, 검정 그라데이션 깊이, 달력 계열 패널 스타일 적용
- `current_task.md`: 이번 작업 보고서로 갱신
